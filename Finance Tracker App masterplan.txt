# Finance Tracker App - Anti-Gravity Master Plan

## Project Overview
**Name:** Finance Tracker App  
**Type:** Cross-platform mobile app (Flutter) + Admin Dashboard (Laravel + React)  
**Purpose:** Portfolio project demonstrating full-stack development, offline-first mobile functionality, cloud sync, and admin dashboard with analytics.

**Goals:**
- Minimalist, fully functional finance tracker  
- Offline-first with optional cloud sync  
- Encrypted local storage  
- Admin dashboard for internal portfolio demo  
- Supabase Auth for login  

---

## Technology Stack
- **Mobile App:** Flutter 3.13+, Dart 3.x  
- **Backend API:** Laravel 10.x, PHP 8.2  
- **Admin Frontend:** React 18 + Tailwind CSS  
- **Database:** Supabase Postgres (encrypted fields)  
- **Auth:** Supabase Auth  
- **Local Storage:** SQLite (encrypted)

## Core Features

### Mobile App (MVP)
- Add, edit, delete income & expense transactions  
- Categorize transactions  
- Offline-first with SQLite, encrypted storage  
- Optional cloud sync with Supabase Postgres  
- Charts: monthly, weekly, by category  
- Supabase Auth login/signup  

### Admin Dashboard (MVP)
- Manage categories/templates  
- View anonymized usage analytics  
- Feature flags management  
- Internal portfolio demo only
