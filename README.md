# Finance Tracker Mobile

A premium, offline-first mobile application for personal finance tracking, built with Flutter. This project focuses on providing a high-end user experience with a modern fintech aesthetic.

- **Premium UI/UX**: Designed with a "WOW" factor using Glassmorphism, Dark Mode, and smooth animations.

- **Glassmorphism**: Beautiful, layered interface with backdrop blur and gradient effects.

## üöÄ Getting Started

### Prerequisites

- Flutter SDK (3.13+)
- Dart SDK
- Android Studio / VS Code with Flutter extension

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/finance-tracker-mobile.git
   ```
2. Navigate to the project directory:
   ```bash
   cd finance-tracker-mobile
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
